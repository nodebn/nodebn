# Final Subscription Plans (Freemium with BND Pricing)

Incorporating your preferred BND pricing (19/49/139) for the paid tiers:

## 0. **Free Plan** - BND 0/month
- **Stores**: 1 store
- **Products**: 10 products per store (with variants & images)
- **Features**: Basic CRUD for categories, services, promo codes, payment methods
- **Branding**: Platform branding (NodeBN logo)
- **Integration**: WhatsApp checkout
- **Support**: Community forum/basic docs
- **Limits**: Standard usage, upgrade prompts

## 1. **Starter Plan** - BND 19/month
- **Stores**: 1 store
- **Products**: 20 products per store
- **Features**: All Free features + custom store logo, remove platform branding
- **Branding**: Custom logo
- **Integration**: WhatsApp checkout with order messaging
- **Support**: Email support within 48 hours
- **Limits**: Standard processing

## 2. **Professional Plan** - BND 49/month
- **Stores**: 3 stores
- **Products**: 100 products per store
- **Features**: All Starter features + bulk uploads, export tools, basic API
- **Branding**: Basic theme customization
- **Integration**: Priority WhatsApp processing
- **Support**: Priority email + chat, monthly check-ins
- **Limits**: Faster processing, advanced validation

## 3. **Enterprise Plan** - BND 139/month
- **Stores**: 10 stores
- **Products**: Unlimited products per store
- **Features**: All Professional features + custom themes, advanced bulk ops, full API
- **Branding**: Advanced customization
- **Integration**: Dedicated WhatsApp processing
- **Support**: Dedicated manager, phone support, quarterly reviews
- **Limits**: Premium speeds, unlimited exports

## Key Features:
- **Freemium Model**: Free tier attracts users, paid tiers provide value
- **BND Pricing**: Local currency for Brunei market
- **Progressive Limits**: Clear upgrade incentives
- **Deliverable**: All features based on your current system

This structure balances acquisition (free) with monetization (paid tiers). The Professional plan at BND 49 offers the best value for growing sellers.</content>
